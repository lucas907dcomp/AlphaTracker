## Description

Bootstrap the AlphaTracker SPA project using Vite + React + TypeScript + Tailwind. Install all 15 production dependencies defined in `implementation.yaml`. Configure TypeScript strict mode with absolute import paths. Produce the `.env.example` file with required environment variable keys.
