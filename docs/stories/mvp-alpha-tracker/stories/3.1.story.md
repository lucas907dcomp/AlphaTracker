## Description

Implement the core PnL calculation engine and date aggregation utilities. `pnl.ts` exports `calcularCenariosPnL` which applies the 4 formulas (Simétrico, Assimétrico, Freebet SNR, Duplo Green) using `decimal.js` for all arithmetic — native JS float operators are **prohibited** (CON-3). `dates.ts` exports period helpers used by `useDashboard` in Wave 6. These are pure functions with zero side effects.

## Technical Notes

- **CON-3 (CRITICAL):** `decimal.js` is mandatory. Never write `a.gross_return - totalCusto`; always write `new Decimal(a.gross_return).minus(totalCusto)`. A single native float operation invalidates financial correctness.
- **`Aposta` type fields (snake_case):** The DB-sourced type uses `gross_return`, `is_freebet`, `is_double_green`, `casa_id`. The spec.md code sample uses camelCase — that sample is illustrative only. Use the actual `src/types/index.ts` field names.
- **`casaNome` from optional join:** `aposta.casa` is `Casa | undefined`. Use `aposta.casa?.nome ?? ''` to handle apostas loaded without the join.
- **`toDecimalPlaces(2).toNumber()`:** The final `pnl` value in `CenarioPnL` is a `number` (2 decimal places) — use `new Decimal(x).toDecimalPlaces(2).toNumber()`.
- **`date-fns` version:** v3.x is installed. Use named imports: `import { startOfDay, startOfWeek, startOfMonth } from 'date-fns'`.
- **Spec ref:** Section 3.2 (PnL Calculation Engine), FR-4, FR-5, FR-7, FR-8, CON-3.

```typescript
// pnl.ts — reference implementation
import Decimal from 'decimal.js'
import type { Aposta, CenarioPnL } from '@/types'

export function calcularCenariosPnL(apostas: Aposta[]): CenarioPnL[] {
  const totalCusto = apostas
    .filter(a => !a.is_freebet)
    .reduce((acc, a) => acc.plus(new Decimal(a.stake)), new Decimal(0))

  return apostas.map(aposta => {
    const gr = aposta.is_double_green
      ? new Decimal(aposta.gross_return).times(2)
      : new Decimal(aposta.gross_return)
    return {
      casaId: aposta.casa_id,
      casaNome: aposta.casa?.nome ?? '',
      pnl: gr.minus(totalCusto).toDecimalPlaces(2).toNumber(),
    }
  })
}
```

```typescript
// dates.ts — reference implementation
import { startOfDay, startOfWeek, startOfMonth, parseISO, isAfter, isEqual } from 'date-fns'

export function getStartOfDay(d: Date): Date {
  return startOfDay(d)
}

export function getStartOfWeek(d: Date): Date {
  return startOfWeek(d, { weekStartsOn: 1 })
}

export function getStartOfMonth(d: Date): Date {
  return startOfMonth(d)
}

export function isInPeriod(operacaoDate: string, period: 'day' | 'week' | 'month'): boolean {
  const date = parseISO(operacaoDate)
  const now = new Date()
  const starts: Record<typeof period, Date> = {
    day: getStartOfDay(now),
    week: getStartOfWeek(now),
    month: getStartOfMonth(now),
  }
  const start = starts[period]
  return isAfter(date, start) || isEqual(date, start)
}
```
