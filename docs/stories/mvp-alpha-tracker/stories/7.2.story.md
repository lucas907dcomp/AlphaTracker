## Description

Implementa o CRUD completo de parceiros e a configuração de vínculo entre casas e parceiros. Inclui: novos tipos TypeScript, hook `useParceiros`, página `/parceiros` e atualização do `CasaForm`/`CasaList` para exibir e configurar o parceiro de cada casa.

  ```typescript
  export interface Parceiro {
  ```

## Technical Notes

- Seguir padrão existente de hooks: `useQuery` + `useMutation` com `@tanstack/react-query`
- `useCasas` query deve fazer join: `.select('*, parceiro:parceiros(*)')` — o Supabase interpreta `parceiro:parceiros(*)` como FK join
- `percentual` deve ser validado: inteiro ou decimal, entre 1 e 99 (o dono recebe o complemento implicitamente)
- Usar componente `Select` existente em `src/components/ui/Select.tsx` para o dropdown de parceiro nas casas
- `CasaForm` (`src/components/casas/CasaForm.tsx`) é para criação — a vinculação de parceiro pode ser inline na lista (`CasaList`) para não quebrar o flow de criação de casa
- Manter o padrão de `zod` para validação de formulário (`src/schemas/`)
