## Description

Build `ResultadoModal` — the component that finalizes an operation by marking which leg won. The user selects the winning aposta from a radio list, then confirms. On confirmation the component calls `marcarResultado` (already implemented in `useOperacoes`), which marks the winner `resultado: 'Ganhou'`, all others `'Perdeu'`, calculates the definitive PnL via `calcularCenariosPnL`, and transitions the operation status `Pendente → Concluida`.

Browser verification: open a Pendente operation, select the winning leg, confirm — the modal closes, the operation becomes Concluida, and the definitive PnL is visible in the list.

## Technical Notes

- **`useOperacoes` inside component:** Call `const { marcarResultado } = useOperacoes()` directly inside the modal — TanStack Query deduplicates the query, so this does NOT trigger an extra fetch.
- **`isSubmitting` local state:** Use `const [isSubmitting, setIsSubmitting] = useState(false)` — not from react-hook-form (no form here).
- **`selectedWinnerId` state:** `const [selectedWinnerId, setSelectedWinnerId] = useState<string | null>(null)`
- **Radio group pattern:**
  ```tsx
  {operacao.apostas!.map((aposta, i) => (
    <label key={aposta.id} className="flex items-center gap-2 cursor-pointer">
      <input
        type="radio"
        name="winner"
        value={aposta.id}
        checked={selectedWinnerId === aposta.id}
        onChange={() => setSelectedWinnerId(aposta.id)}
      />
      <span>
        {aposta.casa?.nome ?? 'Casa'}{aposta.is_freebet ? ' (FB)' : ''}
        {' | '}Stake: R$ {aposta.stake}
        {' | '}Ret: R$ {aposta.is_double_green ? aposta.gross_return * 2 : aposta.gross_return}{aposta.is_double_green ? ' (2x)' : ''}
      </span>
    </label>
  ))}
  ```
- **Confirm handler:**
  ```tsx
  async function handleConfirm() {
    if (!selectedWinnerId) return
    setIsSubmitting(true)
    try {
      await marcarResultado({
        operacaoId: operacao.id,
        winnerApostaId: selectedWinnerId,
        apostas: operacao.apostas!,
      })
      toast.success('Resultado marcado!')
      onClose()
    } catch (error) {
      toast.error(error instanceof Error ? error.message : 'Erro ao marcar resultado')
    } finally {
      setIsSubmitting(false)
    }
  }
  ```
- **Overlay backdrop:** `className="fixed inset-0 bg-black/50 flex items-center justify-center z-50"`
- **Content panel:** `className="bg-white rounded-lg p-6 w-full max-w-md mx-4 space-y-4"`
- **Sonner toast:** `import { toast } from 'sonner'`
- **Types:** `import type { Operacao } from '@/types'`
- **Spec refs:** FR-12 (mark result, Pendente → Concluida), FR-5 (definitive PnL)
