## Description

Assemble the complete operations management view by composing `OperacaoCard`, `OperacaoList`, and `OperacoesPage`:

- **`OperacaoCard.tsx`** — Displays one operation with its status chip, tipo badge, date, legs, and PnL. For `Concluida` operations: shows definitive PnL. For `Pendente`: shows real-time CenarioPnL scenarios. Has a "Marcar Resultado" button for Pendente operations.
- **`OperacaoList.tsx`** — Renders the list from `useOperacoes()` with proper loading and empty states.
- **`OperacoesPage.tsx`** — Replaces the Wave 5 stub. Renders `OperacaoList` + manages `ResultadoModal` state (which operation is open).

Browser verification: list shows operations; opening a Pendente card via "Marcar Resultado" opens the modal; confirming updates the card to Concluida with definitive PnL.

## Technical Notes

- **`calcularCenariosPnL` in OperacaoCard:** `import { calcularCenariosPnL } from '@/lib/pnl'`. Called directly in the render path with `operacao.apostas`. Since `calcularCenariosPnL` is pure and fast (< 1ms), no memoization needed for MVP.
- **`Operacao.apostas` availability:** The `useOperacoes` query uses `select('*, apostas(*, casa:casas(*))')` — apostas are always present in the result. Safe to access `operacao.apostas!` for type assertion, but defend against undefined with optional chaining in the template.
- **Date formatting:** Appending `T00:00:00` prevents the date from shifting by one day due to UTC interpretation:
  ```tsx
  new Date(operacao.data + 'T00:00:00').toLocaleDateString('pt-BR')
  // output: "18/05/2026" instead of potentially "17/05/2026"
  ```
- **Tipo badge colors** (Tailwind, minimal set for MVP):
  ```tsx
  const tipoBadgeColor: Record<OperacaoTipo, string> = {
    Freebet: 'bg-purple-100 text-purple-800',
    Extracao: 'bg-blue-100 text-blue-800',
    SuperOdd: 'bg-orange-100 text-orange-800',
    TentativaDuplo: 'bg-yellow-100 text-yellow-800',
    Aposta: 'bg-gray-100 text-gray-800',
  }
  ```
- **Status chip colors:**
  - Pendente: `bg-yellow-100 text-yellow-800`
  - Concluida: `bg-green-100 text-green-800`
- **PnL color:** `pnl > 0 ? 'text-green-600' : 'text-red-600'`
- **PnL format:** `R$ ${pnl.toFixed(2).replace('.', ',')}` — Brazilian Real format
- **OperacaoCard container:** `border border-gray-200 rounded-lg p-4 space-y-3`
- **`Link` for Nova Operação:** `import { Link } from 'react-router-dom'` → `<Link to="/operacoes/nova">Nova Operação</Link>` styled as a Button variant="primary"
- **Import types:** `import type { Operacao } from '@/types'` + `import type { OperacaoTipo } from '@/types'`
- **Spec refs:** FR-12 (mark result, Pendente → Concluida), FR-5 (PnL calculation), FR-8 (isDoubleGreen 2x badge), FR-9 (visible PnL per operation)
