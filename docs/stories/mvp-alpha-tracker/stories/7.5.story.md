## Description

Finaliza a Wave 7 com a visibilidade financeira do sistema de parceiros. Adiciona ao Dashboard um widget de saldo por parceiro e uma tela de registro de repasses (pagamentos feitos ao parceiro).

O saldo devedor é calculado como:
```
saldo = Σ(parte_parceiro de todas as operações Concluidas com parceiro) - Σ(repasses pagos)
```

O usuário pode registrar um repasse (pagamento ao parceiro) que desconta do saldo acumulado.

## Technical Notes

- O cálculo de saldo é feito **no frontend** — não há view ou função no Supabase. Isso mantém a simplicidade e é suficiente para o volume de dados esperado
- A query de operações para cálculo de saldo deve filtrar por `status = 'Concluida'` E `(custo_liberacao IS NOT NULL OR operacao_origem_id IS NOT NULL)` para pegar apenas operações com split configurado
- Se `lucroLiquido < 0` (operação deu prejuízo líquido), `parteParceiro` também é negativa — isso reduz o saldo devedor (raro mas possível)
- `ParceiroBalanceCard` deve usar `useParceiroDashboard` com `queryKey: ['parceiro-dashboard']` para revalidar após repasses
- Usar `CentavosInput` para o campo `valor` no `RepasseModal`
- Manter consistência visual com `PnLSummaryCard` existente em `src/components/dashboard/PnLSummaryCard.tsx`
